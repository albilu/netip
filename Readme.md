***NETIP 3.0***

	Netip is a command line IP and IPLookup tool.

***ENVIRONEMENT***
	
	This tool need curl and lynx to work properly ! So install them if ask !
	GNU/LINUX

***COMMAND***

	# netip without option for IPLookup of your own machine
	# netip ip_adress
	# netip --help

***INSTALLATION***
	
	1-Extract archive
	2-cd path_to../Netip	
	3-sudo ./Netip.run
